System architecture

Five-stage inference pipeline

Every step runs locally on the device. No audio ever leaves the hardware.

01
Audio capture

Mic input locked at 16 kHz, mono channel. 10-second recording window with real-time amplitude visualization. Leading/trailing silence auto-trimmed using librosa energy threshold (top_db = 20).

16 kHz WAV Flutter record
02
Spectral denoising

First 500ms treated as noise floor sample. Spectral gate applied: STFT mask where signal power < 2× noise power is zeroed. Reconstructed via iSTFT. Dramatically reduces ambient environmental noise.

Spectral gating STFT/iSTFT Librosa
03
Mel-spectrogram transformation

128 Mel filterbanks, n_fft = 2048, hop = 512. Frequency range 50–8000 Hz captures all clinically relevant respiratory harmonics. Power converted to dB scale (log compression). Output: 224×224 PNG.

128 Mel bins 224×224 Log-scale dB
04
CNN inference (TFLite)

MobileNetV2 backbone fine-tuned on COUGHVID + Coswara. INT8 quantized via representative dataset calibration. Runs entirely in RAM — zero disk I/O during inference. Avg latency < 50ms on mid-range ARM.

MobileNetV2 INT8 tflite_flutter
05
Risk stratification & report

Softmax output mapped to four classes with probability scores. Primary label + confidence displayed. Full report (timestamp, audio path, class probabilities) written to local SQLite. Zero network calls at any stage.

Softmax 4-class SQLite Offline-first
Technology

Built on proven tools

🎵
Librosa
Audio processing

STFT, Mel-filterbank computation, silence trimming, and spectral gating. Industry standard for audio ML preprocessing.

🧠
TensorFlow / PyTorch
Model training

MobileNetV2 transfer learning with two-stage fine-tuning. Categorical cross-entropy, AUC tracked to handle class imbalance.

⚡
TensorFlow Lite
On-device inference

INT8 post-training quantization. 4× size reduction, 2–3× speed gain. Runs on any mid-range Android/iOS via NNAPI/CoreML.

📱
Flutter + Dart
Cross-platform UI

Single codebase targeting Android and iOS. Native audio APIs via the record package. tflite_flutter for model inference binding.

🗄️
SQLite (sqflite)
Local storage

All triage reports stored locally. Schema captures timestamp, audio path, and four class probability scores per report.

📊
COUGHVID + Coswara
Training data

EPFL's 25k-sample COUGHVID and IISc's Project Coswara provide labeled respiratory audio across multiple conditions.
